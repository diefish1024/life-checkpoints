# 2025 年终总结

坐在期末考的复习堆里写总结，心境和去年截然不同。去年这时候，我还在生医工的各种大饼里焦虑，想着考高分转专业；今年此时，虽然还在为 John 班的数学课头秃，但至少身份已经是 CS 学生了。

轻舟已过万重山，但这山过得也没那么潇洒，更多的是跌跌撞撞。

## 关于转轨与祛魅

回想年初发那个转专业咨询帖，现在看完全是处于“愚昧之巅”。为了显得自己很强，硬凑了几个项目经历，实际上连 Git 都不会用。直到有学长真的来交流项目，我才连夜把手写的 STL 传上 GitHub，那种汗流浃背的感觉到现在还记得。但也感谢那时的冲动，让我结识了不少厉害的学长，让我认真思考了转入致远计算机的可能性。

上半年其实还花了不少精力在实验室，搞脑机接口。前后换了两个项目，折腾了半天最终也没搞出什么名堂。直到后来进了超算队，两边实在顾不过来，就索性退出了实验室。这段经历虽然没什么产出，但也让我把想走的路想清楚了：我不适合也不喜欢科研。相比于在学术圈里发 Paper，我还是更想直接工作，去工业界写点能跑的代码。

转专业的过程也是对自己能力的祛魅。借助 AI 磕磕绊绊速通了几个 ACM 班大作业，又试图手写一个数据库，在堆了 2k 行代码的屎山后，我彻底认清了现实，再写下去要被金课肘飞了。好在运气不错，四月致远发布校内招生通知，虽然面试发挥不算完美，但今年政策宽容，最终还是顺利转入了 John Class。这一步迈过去，心里那块大石头总算是落地了，以后可以名正言顺地学自己想学的东西了。

## 关于技术

转专业后，为了找回学习状态，暑假啃完了 NJU-OS。下半年拿了 HelloHPC 校内赛第一，进了超算队，国赛也拿了奖。

听起来挺顺，但我自己心里其实很虚。

这一年，我越来越依赖 Vibe Coding。很多时候，代码是靠 AI 生成，逻辑是靠“感觉”跑通的。那个 HelloHPC 的第一名，有多少成分是属于我自己的实力，又有多少是属于 AI 的？这种“只有产出，没有过程”的状态让我很焦虑。

现在的真实状态是，如果不依赖 AI，我大概率写不出足够健壮的代码，甚至可能连基本的实现都会卡壳。我不确定那种脱离 Copilot 一行行敲代码的能力是否还算“工匠精神”，但我知道，现在这种状态肯定是不健康的。

明年没什么宏大的技术愿景，只希望把工具链用得更熟练，把代码写得更扎实一点。如果谈的更具体一点，大概想在 AI Infra 上钻地更深一些，对其他方向比如 Agent 也多探索探索，多和厉害的人交流，多看看多学学总是没有错的。

## 关于生活

以前觉得生活是用来填补工作学习间隙的，今年觉得反过来了。

感情上，上半年在内耗中度过，好在暑假下定决心分了手。那段经历让我明白，坦率是解决错位的唯一解药。带着这点长进，10 月份很幸运地遇到了现在女朋友。两个人在深秋确认关系，在初冬一起自习，一起出游。

除了恋爱，今年也开始在“折腾”自己中找乐子。5 月份心血来潮学了贝斯，虽然练了三个月就因为开学太忙搁置了，但也算是从头开始捣鼓了一种乐器。9 月开始健身，看着镜子里的线条慢慢变清晰，算是一种难得的“付出就有回报”的反馈。12 月跟着野协走了一条重装线，背着重装包在大山里走得气喘吁吁，但获得的成就感也是格外踏实的，明年如果有空，还想再去走几条新线。

## 写在最后

2025 这一年，没成什么很厉害的人，也没做什么惊天动地的大事。

我只是从一个焦虑的 BME 学生变成了一个焦虑的 CS 学生，顺便谈了一场开心的恋爱，去山里走了一趟路，并确定了自己以后要找个班上。

对于 2026，希望 John 班的数学课能对我好一点，希望能顺利找到一份实习。更希望在 AI 的洪流里，能做一个稍微像样点的工程师。
